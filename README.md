### Сейчас в фокусе

Развиваю **Fair Splitter**, сервис для честного деления расходов между людьми. Считаю счёт по позициям из чека, по долям и пропорционально доходу, свожу все долги к минимальному числу переводов.

Параллельно:

- Backend и API на **Python, Django, DRF** поверх **PostgreSQL**.
- Мобильные приложения на **Flutter**, веб на **Next.js** и **React**.
- ML-задачи: предсказательные модели, кластеризация и обучение с подкреплением.
